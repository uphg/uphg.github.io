Cline MCP 入门指南

发布于 2025-03-30

前置条件 ​

  • ✅ 最新版本的 Node.js（v18 或更高）
    • 检查方法：运行 node --version
    • 安装来源：https://nodejs.org/
  • ✅ 最新版本的 Python（v3.8 或更高）
    • 检查方法：运行 python --version
    • 安装来源：https://python.org/
  • ✅ UV 包管理器
    • 在安装 Python 后，运行：pip install uv
    • 验证版本：uv --version

各类型服务器安装 ​

Filesystem 服务器 ​

bash
npm install -g @modelcontextprotocol/server-filesystem

Playwright 服务器 ​

bash
npm install -g @executeautomation/playwright-mcp-server

Git 服务器 ​

bash
npm install -g @modelcontextprotocol/server-git

统一配置方法 ​

创建配置文件 cline_mcp_settings.json:

json
{
  "mcpServers": {
    "filesystem": {
      "command": "node",
      "args": ["C:\\Users\\UserName\\AppData\\Roaming\\npm\\node_modules\\@modelcontextprotocol\\server-filesystem\\dist\\index.js", "F:\\Development\\Personal\\blog-creator"],
      "env": {},
      "disabled": false,
      "autoApprove": []
    },
    "playwright": {
      "command": "node",
      "args": ["C:\\Users\\UserName\\AppData\\Roaming\\npm\\node_modules\\@executeautomation\\playwright-mcp-server\\dist\\index.js"],
      "disabled": false,
      "autoApprove": []
    },
    "git": {
      "command": "uvx",
      "args": ["mcp-server-git", "--repository", "F:\\Development\\Personal\\blog-creator"],
      "disabled": false,
      "autoApprove": []
    }
  }
}

启动 MCP 服务器 ​

配置文件会自动加载，服务器将在后台运行。

测试 MCP 服务器 ​

注意

第一次安装服务器，测试前需要重启 IDE。

使用以下命令测试服务器是否正常工作：

bash
# 列出允许访问的目录
curl -X POST http://localhost:8080/list_allowed_directories

# 列出目录内容  
curl -X POST http://localhost:8080/list_directory -H "Content-Type: application/json" -d '{"path":"F:/Development/Personal/blog-creator"}'

常见问题 ​

路径访问被拒绝 ​

确保路径大小写完全匹配，如使用 F: 而不是 f:

权限问题 ​

检查配置文件中的路径是否有访问权限
