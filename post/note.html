CSS 常用语法 ​

媒体查询 ​

表示最小宽度为 100px 最大宽度为 200px 的设备：

css
@media (min-width: 100px) and (max-width: 200px) {
  body {
    属性: 值;
  }
}
@media (min-width: 100px) and (max-width: 200px) {
  body {
    属性: 值;
  }
}

设置 PDF 样式 ​

在 @media print 中的样式只会在 PDF 格式下生效

css
@media print {
  body {
    background: red;
  }
}
@media print {
  body {
    background: red;
  }
}

transform 动画 ​

移动位置括号内表示坐标 (X 轴 Y 轴)，单位可以使用百分比 ( 百分比根据自身 )

css

放大缩小括号内表示坐标 (X 轴 Y 轴)

css
transform: scale(2, 2); /* 放大 */
transform: scale(0.3, 0.3); /* 缩小 */
transform: scale(2, 2); /* 放大 */

CSS 变量 ​

以设置字体变量为例：

css
/* 声明变量 */
:root {
  --font-size: 14px;
}
/* 使用变量 */
p {
  font-size: var(--font-size);
}
/* 声明变量 */
:root {
  --font-size: 14px;
}
/* 使用变量 */
p {
  font-size: var(--font-size);
}

提示

:root 表示作用域，可以改为 html，或上面案例中的 p 标签

计算属性 ​

计算属性可以让 CSS 支持支持 +、-、*、/ 运算

css
/* 语法 property: calc(expression) */
width: calc(100% - 80px);
/* 语法 property: calc(expression) */
width: calc(100% - 80px);

注意

+ 和 - 运算符的两边必须要有空白字符。

* 和 / 这两个运算符前后不需要空白字符，但如果考虑到统一性，仍然推荐加上空白符。

设置编码格式 ​

css

@charset "UTF-8";

引入其他文件 ​

@import 是引入其他 CSS 文件

css
@import url(style.css);
@import url(style.css);

注意

@charset 和 @import 结尾必须加分号

@import 的性能较低，通常推荐使用 JS 中的模块化（import 'style.css'）引入

连续字母换行 ​

有时多个连续的英文字母即时用内联元素包裹也无法换行，如下：

html
<div><span>ccccccccccccccccccccccccccccccc</span></div>
<div><span>ccccccccccccccccccccccccccccccc</span></div>

这时给 div 添加以下属性

css
div {
  word-break: break-all;
}
div {
  word-break: break-all;
}

连续字母就可以换行了

效果链接：http://js.jirengu.com/hehatujuwe/1/edit

解决定位时文字竖直排列 ​

当一个内联元素绝对定位时，会竖直排列，而不是正常横向排列给内联元素添加以下属性即可

css
p {
  white-space: nowrap;
}
p {
  white-space: nowrap;
}

使 pre 标签自动换行 ​

默认情况下，在 pre 标签中的内容是不会自动换行的，添加以下属性即可像其他元素一样实现根据元素宽度换行。

css
pre {
  white-space: pre-wrap;
  white-space: -moz-pre-wrap;
  white-space: -pre-wrap;
  white-space: -o-pre-wrap;
  word-wrap: break-word;
}
pre {
  white-space: pre-wrap;
  white-space: -moz-pre-wrap;
  white-space: -pre-wrap;
  white-space: -o-pre-wrap;
  word-wrap: break-word;
}
