CSS 变量

发布于 2021-11-12

声明变量 ​

CSS 变量声明时，前面要加 --

css
body {
  --foo: #000000;
  --bar: #F7EFD2;
}

各种值都可以放入 CSS 变量。

css
:root{
  --main-color: #4d4e53;
  --main-bg: rgb(255, 255, 255);
  --logo-border-color: rebeccapurple;

  --header-height: 68px;
  --content-padding: 10px 20px;

  --base-line-height: 1.428571429;
  --transition-duration: .35s;
  --external-link: "external link";
  --margin-top: calc(2vh + 20px);
}

注意

变量名大小写敏感，--header-color 和 --Header-Color 是两个不同变量。

读取变量 ​

使用 var() 函数读取变量

css
:root {
  --primary-color: red;
  --text-color: var(--primary-color);
}

a {
  color: var(--foo);
  text-decoration-color: var(--bar);
}

变量默认值 ​

var() 函数还可以使用第二个参数，表示变量的默认值。如果该变量不存在，就会使用这个默认值（仅限于变量未定义时使用）。

css
:root {
  color: var(--foo, #7F583F);
}

第二个参数后的逗号和空格都不会处理，例如以下字体声明：

css
body {
  font-family: var(--font-stack, "Roboto", "宋体", "楷体");
}

当 --font-stack 变量不存在时，它真实的值会变为：

css
body {
  font-family: "Roboto", "宋体", "楷体"
}

如果变量值不合法，会使用继承值或初始值代替，而不是 var() 中的默认值，例如以下代码：

css
body {
  --color: 20px;
  background-color: #369;
  background-color: var(--color, #cd0000);
}

它的实际值会变成：

css
body {
  --color: 20px;
  background-color: #369;
  background-color: transparent;
}

变量值的类型 ​

当变量值是一个字符串时，可以与其他字符串拼接。

css
:root {
  --bar: 'hello';
  --foo: var(--bar)' world';
}

如果变量值是数值，不能与数值单位直接连用。

css
.foo {
  --gap: 20;
  /* 以下方式无效 */
  margin-top: var(--gap)px;
}

必须使用 calc() 函数，将它们连接。

css
.foo {
  --gap: 20;
  margin-top: calc(var(--gap) * 1px);
}

或者更复杂的使用 CSS3 calc() 计算，例如：

css
body {
  --columns: 4;
  --margins: calc(24px / var(--columns));
}

如果变量值带有单位，就不能写成字符串。

css
/* 无效 */
.foo {
  --foo: '20px';
  font-size: var(--foo);
}

/* 有效 */
.foo {
  --foo: 20px;
  font-size: var(--foo);
}

作用域 ​

同一个 CSS 变量，可以在多个选择器内声明。读取的时候，优先级最高的声明生效。这与 CSS 的"层叠"（cascade）规则是一致的。

下面是一个例子。

html
<style>
  :root { --color: blue; }
  div { --color: green; }
  #alert { --color: red; }
  * { color: var(--color); }
</style>

<p>蓝色</p>
<div>绿色</div>
<div id="alert">红色</div>

因此，全局的变量通常放在根元素:root里面，确保任何选择器都可以读取它们。

css
:root {
  --main-color: #06c;
}

响应式布局 ​

CSS 是动态的，页面的任何变化，都会导致采用的规则变化。

利用这个特点，可以在响应式布局的 media 命令里面声明变量，使得不同的屏幕宽度有不同的变量值。

css
body {
  doc-padding: 32px 24px 96px;
  padding: var(--doc-padding);
}

@media (min-width: 960px) {
  body {
    doc-padding: 48px 32px 0;
  }
}

@media (min-width: 768px) {
  body {
    doc-padding: 48px 32px 128px;
  }
}

兼容性处理 ​

假如浏览器不支持 CSS 变量，可以使用 @support 命令进行检测。这被称为特性查询。

css
@supports ( (--foo: 0)) {
  /* supported */
}

@supports ( not (--foo: 0)) {
  /* not supported */
}

也可以使用 JavaScript 检测浏览器是否支持 CSS 变量，方法如下：

javascript
const isSupported =
  window.CSS &&
  window.CSS.supports &&
  window.CSS.supports('--a', 0);

if (isSupported) {
  /* supported */
} else {
  /* not supported */
}

JavaScript 操作 CSS 变量 ​

简单的操作

javascript
// 设置变量
document.body.style.setProperty('--primary', '#7F583F');

// 读取变量
document.body.style.getPropertyValue('--primary').trim();
// '#7F583F'

// 删除变量
document.body.style.removeProperty('--primary');

这意味着，JavaScript 可以将任意值存入样式表。下面是一个监听事件的例子，事件信息被存入 CSS 变量。

javascript
const docStyle = document.documentElement.style;

document.addEventListener('mousemove', (e) => {
  docStyle.setProperty('--mouse-x', e.clientX);
  docStyle.setProperty('--mouse-y', e.clientY);
});

那些对 CSS 无用的信息，也可以放入 CSS 变量。

css
--foo: if(x > 5) this.width = 10;

上面代码中，--foo 的值在 CSS 里面是无效语句，但是可以被 JavaScript 读取。这意味着，可以把样式设置写在 CSS 变量中，让 JavaScript 读取。

所以，CSS 变量提供了 JavaScript 与 CSS 通信的一种途径。

变量命名规范 ​

默认情况下，使用"名称 + 属性"的方式

css
body {
  --button-padding: 16px;
  --button-border-color: #C0C4CC;
  --navbar-text-color: #000000;
  --search-bg-color: #ffffff;
}

如果附带类型，比如大小，可以使用"名称 + 类型 + 属性"

css
body {
  --button-height: 36px;
  --button-small-height: 32px;
  --button-big-height: 40px;
}
