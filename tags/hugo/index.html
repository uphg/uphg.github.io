Hugo

Hugo 搭建个人博客

此教程只适用于在 Windows10 环境下搭建。其他系统在官网自行查询教程即可。 安装 去Github下载：https://github.com/gohugoio/hugo/releases 通常选择 hugo_X.XX.X_Windows-64bit.zip 或者 hugo_extended_X.XX.X_Windows-64bit.zip（拓展版）即可，部分博客可能需要安装拓展版。 下载后最好解压到一个无空格无中文的目录。 解压后在我的电脑中配置环境，分别按照以下步骤点击： 右键此电脑 > 属性 > 高级系统设置 > 环境变量 > 选择「系统变量」下的 Path 编辑，在 Path 中添加 hugo 的安装路径。
